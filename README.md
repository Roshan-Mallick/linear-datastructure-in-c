## 🚀 Getting Started

### Prerequisites
- GCC compiler
- Linux / macOS / WSL

### Compile & Run

```bash
# Clone the repo
git clone https://github.com/Roshan-Mallick/linear-datastructure-in-c.git
cd linear-datastructure-in-c

# Compile any file
gcc 01_arrays/01_insert_element.c -o insert_element

# Run it
./insert_element
```
